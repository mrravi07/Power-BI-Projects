# 📖 Overview

The Ravi E-Commerce Sales Dashboard is an interactive Business Intelligence solution developed in Power BI to monitor sales performance, profitability, customer behavior, and category-level trends.

The dashboard provides executives and business stakeholders with a centralized view of key performance indicators, enabling faster decision-making through data visualization and interactive filtering.

---

# 🎯 Business Objectives

- Monitor overall sales performance
- Track revenue and profit trends
- Analyze customer purchasing behavior
- Evaluate category-level performance
- Identify top-performing products
- Support business decision-making through visual analytics

---

# 📊 Key Performance Indicators

The dashboard provides visibility into:

- Total Sales Amount
- Order Quantity
- Total Profit
- Average Order Value (AOV)
- Monthly Profit Trends
- Category Performance
- Customer Performance
- Payment Method Analysis
